# Task Management App

A simple Task Management application built with **React**, **Node.js**, **Express**, and **MongoDB**.  
Users can **create, update, delete, and filter tasks** based on their status: Pending, In-Progress, or Done.

## Features
- Add new tasks with title and description
- Edit existing tasks
- Delete tasks
- **Filter tasks** by status (Pending, In-Progress, Done)
- Responsive and clean UI using Tailwind CSS
- 
## Tech Stack

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- API calls using `fetch`
